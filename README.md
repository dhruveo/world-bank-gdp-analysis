# Economic Data Analysis Using the World Bank API and Python

## Project Overview

This project analyzes India's Gross Domestic Product (GDP) using the World Bank Public API. The data is collected, stored in CSV format, analyzed in Google Colab, and visualized using Python.

## Tools Used

- Python
- Google Colab
- Pandas
- Matplotlib
- World Bank API

## Files

- World_Bank_GDP_Analysis.ipynb
- india_gdp.csv
- requirements.txt

## Visualizations

- GDP Trend Over Time
- Top 10 Years with Highest GDP
- GDP Distribution Histogram
- GDP Box Plot
- GDP Scatter Plot
